The script is one possible solution of the following exercise: 

2. Add and Subtract
You will receive three integer numbers.
Write a function sum_numbers() to get the sum of the first two integers and subtract() function that subtracts
the third integer from the result. Wrap the two functions in a function called add_and_subtract() which will
receive the three numbers
