The script is one possible solution of the following exercise: 

3. Elevator
Calculate how many courses will be needed to elevate n persons by using an elevator with capacity of p persons.
The input holds two lines: the number of people n and the capacity p of the elevator.
Examples
Input 	Output 	Comments
17	6 	5 courses * 3 people
3		+ 1 course * 2 persons

4	1 	All the persons fit inside the elevator.
5		Only one course is needed.

10	2 	2 courses * 5 people
5
