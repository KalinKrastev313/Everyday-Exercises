The script is one possible solution of the following exercise: 

1. Which Are In?
Given two lists of strings print a new list of the strings that contains words from the first list which are substrings of
any of the strings in the second list (only unique values)
Input
There will be 2 lines of input: the two lists separated by &quot;, &quot;
Output
Print the resulting list on the console

Example
Input 						Output

arp, live, strong				['arp', 'live', 'strong']
lively, alive, harp, sharp, armstrong
