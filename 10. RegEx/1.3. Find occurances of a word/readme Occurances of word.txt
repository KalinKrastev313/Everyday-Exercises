The script is one possible solution of the following exercise: 

Find Occurrences of Word in Sentence
Write a program that finds, how many times a given word, is used in a given sentence. Note that letter case does not matter â€“ it is case-insensitive.
The output is a single number indicating the amount of times the sentence contains the word.
