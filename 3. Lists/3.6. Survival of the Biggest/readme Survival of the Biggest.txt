The script is one possible solution of the following exercise: 

6. Survival of the Biggest
Write a program that receives a list of integer numbers and a number n. The number n represents the amount of
numbers to remove from the list. You should remove the smallest ones.
Input
On the first line you will receive a string (numbers separated by a space), on the second line you will receive a
number n (count of numbers to remove).
Output
Print all the numbers that are left in the list.
Example
Input 		Output
10 9 8 7 6 5
3 		[10, 9, 8]
1 10 2 9 3 8
2 		[10, 9, 3, 8]
